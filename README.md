# 🎬 한국 영화·드라마 퀴즈 게임

터미널에서 동작하는 나만의 퀴즈 게임입니다.  
한국 영화와 드라마에 관한 퀴즈를 풀고, 직접 퀴즈를 추가하며, 최고 점수를 관리할 수 있습니다.


## 📖 프로젝트 개요

Python 기본 문법(클래스, 파일 입출력, 예외 처리)을 활용하여  
콘솔 기반의 퀴즈 게임을 구현한 프로젝트입니다.  
프로그램을 종료하고 재실행해도 퀴즈 데이터와 최고 점수가 유지됩니다.

---

## 🎯 퀴즈 주제 선정 이유

한국 영화와 드라마는 세계적으로 주목받고 있는 콘텐츠입니다.  
'기생충', '오징어 게임', '이상한 변호사 우영우' 등 익숙하면서도 깊이 있는 작품들을 주제로 삼으면  
퀴즈를 푸는 과정이 단순한 암기가 아닌 즐거운 경험이 될 것이라 생각했습니다.


## ✅ 기능 목록

| 기능 | 설명 |
|---|---|
| 퀴즈 풀기 | 등록된 퀴즈를 순서대로 풀고 결과를 확인합니다 |
| 퀴즈 추가 | 문제·선택지 4개·정답 번호를 입력하여 새 퀴즈를 등록합니다 |
| 퀴즈 목록 | 현재 등록된 모든 퀴즈의 제목을 확인합니다 |
| 점수 확인 | 지금까지의 최고 점수를 확인합니다 |
| 데이터 영속성 | 퀴즈와 점수를 `state.json`에 저장하여 재실행 후에도 유지합니다 |
| 예외 처리 | 잘못된 입력, 파일 손상, Ctrl+C 등을 안전하게 처리합니다 |

## 🗂 데이터 파일 설명 (`state.json`)

- **경로:** 프로젝트 루트 `/state.json`
- **인코딩:** UTF-8
- **역할:** 퀴즈 목록과 최고 점수를 영속적으로 저장합니다. 프로그램 시작 시 자동으로 불러오고, 퀴즈 추가·게임 종료 시 자동으로 저장합니다.

### 스키마

```json
{
    "quizzes": [
        {
            "question": "문제 텍스트",
            "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
            "answer": 1
        }
    ],
    "best_score": 80,
    "best_correct": 4,
    "best_total": 5
}
```

| 필드 | 타입 | 설명 |
|---|---|---|
| `quizzes` | array | 퀴즈 객체의 목록 |
| `quizzes[].question` | string | 퀴즈 문제 |
| `quizzes[].choices` | string[] | 4개의 선택지 |
| `quizzes[].answer` | integer (1~4) | 정답 번호 |
| `best_score` | integer | 최고 점수 (백분율) |
| `best_correct` | integer | 최고 점수 당시 정답 수 |
| `best_total` | integer | 최고 점수 당시 총 문제 수 |

> 파일이 없으면 기본 퀴즈 7개로 자동 시작합니다.  
> 파일이 손상된 경우 안내 메시지 출력 후 기본 데이터로 초기화됩니다.
